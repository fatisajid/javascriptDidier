STRUCTURES ITERATIVES

FOR

syntaxe : 

    
        for(initialisation; condition; incrémentation){
            bloc d'instruction...
        }

WHILE

Syntaxe : 

        
            while(condition){
                bloc d'instruction...
            }

comme pour la boucle FOR, tant que la condition est vrai, le bloc d'instruction est exécuté.

la sortie de la boucle n'est pas forcément lié au nombre de tours de boucles.

DO WHILE

Syntaxe : 

    
        do{
            bloc d'instruction...
        } while (condition)

la seul différence avec la boucle WHILE, c'est que la condition est évaluée aprés le bloc d'instruction.

donc, avec une boucle DO WHILE, quelle que soit la condition, il y a au moins 1 tour de boucle qui est effectué avec la boucle WHILE, si la condition est fausse dés le début, on n'effectue aucun tour.

FOR... OF

Boucle qui permet de parcourir un tableau

Syntaxe : 


For(let valeur of nomVariableTableau) {
     bloc d'instruction
}

La boucle FOR... OF permet de parcourir un tableau pour récupérer ses valeurs. A chaque tour de boucle, la variable valeur contient la prochaine valeur du tableau

⚠ on ne peut pas récupérer les indices dans cette boucle
