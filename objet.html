Les OBJETS

Les objets sont des variables complexes comparées aux variables de type simple(string, number, boolean).

Par exemple, les tableaux sont de type object et permettent de stocker plusieurs valeurs dans une seule variable.

Donc un objet peut rassembler plusieurs valeurs. Ces valeurs sont appelés des propriétés(ou attributs).

Ces prpriétés ont un nom et une valeur, comme les variables. Mais ces variablessont liées a un objet. Les propriétéssont les caractéristique de l'objet.

Un objet peut aussi avoir des méthodess : ce sont des fonction qui sont liés a un objet. Ces fonctions ne peuvent etre appelés que par l'intérmédiaire d'un objet.

JSON

JavaScript Objet Notation

Le format JSON utilise la notation des objets javascript pour échanger des données entre différents systémes (serveur, API, navigateur, application,...)

La syntaxe est la suivante : 

            
                champ1: valeur1,
                champ2: valeur2,
                ...
                champN: valeurN,
  • il n'y a que des propriétés, pas de méthodess (car le JSON est utilisé pour transférer des données)
  • Les valeurs de type string doivent etre entourées de "" ou ''
  • attention contrairement a JS, les champs doivent etre entourées de "" ou '' (pour un probléme de comatibilité avec certains systémes)
